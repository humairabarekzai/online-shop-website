# 🎁 Gift Shop - Berlin

A modern, responsive e-commerce front-end application built with HTML, CSS, and Vanilla JavaScript. This project was designed to showcase a seamless shopping experience for a luxury gift boutique based in Berlin.

## ● Project Description
The **Gift Shop** is a beginner-friendly web application that allows users to browse various gift categories, search for specific items, and manage a shopping cart. The UI features a sophisticated dark-themed aesthetic with a custom "Chocolate" color palette, and includes a light-mode toggle for user preference.

## ● Features Included
- **Dynamic Product Rendering:** Products are generated dynamically from a JavaScript array, making it easy to manage inventory.
- **Search & Filter:** - Real-time search bar functionality.
  - Category-based filtering (Cake, Flowers, Watches, etc.).
- **Interactive Shopping Cart:**
  - Add products to the cart with a single click.
  - Increase/decrease item quantities directly in the cart.
  - Real-time total price calculation.
- **Persistent Data:** Uses `localStorage` to ensure the user's shopping cart persists even after a page refresh.
- **Responsive UI:** Built with **Bootstrap 5**, ensuring a smooth experience on mobile, tablet, and desktop devices.
- **Theme Toggle:** Switch between a dark "Luxury" mode and a clean "Light" mode.
- **Product Details:** Bootstrap modals provide a focused view of individual product information.
